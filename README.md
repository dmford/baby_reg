# baby_reg

## Overview
A simple Python script that simulates data, runs a linear regression, and outputs results and a visualization. This project is intended as a clean introduction to regression workflows and reproducible analysis.

## Outputs
- Tables: `./tables/`
- Figures: `./figures/`

## How to Run
- Run the script from the project folder:
```bash
python/baby_reg.py
```

## Dependencies
- numpy
- pandas
- matplotlib
- statsmodels

## Notes
This is a minimal, educational implementation focused on clean structure, reproducibility, and basic statistical interpretation.
